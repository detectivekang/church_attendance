⚠️ 1회용 마이그레이션 페이지

이 페이지는 기존 서산성결교회 데이터를 새 멀티테넌시(다중 교회) 구조로 옮기는 용도입니다. 실행 전 반드시 Firestore 규칙을 아래처럼 임시로 완화해두세요 (마이그레이션 끝나면 즉시 최종 firestore.rules로 교체): 

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}

관리자 계정(kangseabich@naver.com)으로 로그인된 상태여야 합니다.
